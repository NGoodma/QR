# Telegram QR Bot

Простой Telegram-бот, который принимает ссылку и отправляет обратно QR-код в PNG.

## Что умеет
- Команда `/start` с инструкцией.
- Команда `/help` с подсказками.
- Принимает текст с ссылкой (`http://...` или `https://...`) и возвращает QR-код.

## Установка
1. Убедись, что установлен Python 3.10+.
2. Создай виртуальное окружение:

```bash
python -m venv .venv
```

3. Активируй окружение:

```bash
source .venv/bin/activate
```

4. Установи зависимости:

```bash
pip install -r requirements.txt
```

## Когда и где прописывать API-ключ Telegram
Токен бота нужно указать в файле `.env` **перед запуском** приложения.

### Заполни `.env`
```bash
TELEGRAM_BOT_TOKEN="сюда_вставь_токен_от_BotFather"
```

Если переменная не задана, бот завершится с понятной ошибкой.

## Запуск
```bash
python bot.py
```

## Как пользоваться
1. Открой бота в Telegram.
2. Отправь команду `/start`.
3. Отправь ссылку, например: `https://github.com`.
4. Получи изображение QR-кода в ответ.
